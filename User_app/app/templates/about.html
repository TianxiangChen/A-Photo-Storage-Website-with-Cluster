This is the assignment2 for ECE1779 Introduction to Cloud computing.
We tried to excersize as much in this assigment, by not using templates that can be downloaded from the internet. Although we used Bootstrap CSS files, we constructed every page ourselves. This website supports user sign up and upload images. The image uploaded will be processed by the backend to generate 3 transformations, that are flipping, level change and color shifting. The transformations can be seen by click on the thumbnail in the gallery (requires account). The displaying of the modal overlay took much time to perfect. The idea was, showing transformed image alongside with the original one is not elegant, and we wanted to show them once you clicked on it.
We still need to add a function for user to delete the images.

We really wish you liked my design and work. Thanks for your time!
